## 💻 Featured Engineering Highlights

> [!TIP]
> ### 🤖 AI, Deep Learning & Intelligent Automation

### 🚥 Urban Signal — *Intelligent Traffic Optimization Platform*
* **The Core Problem:** Traditional timers cause unnecessary congestion, and emergency vehicles lose critical windows waiting on fixed sequences.
* **The Engineering Solution:** Built an active vision pipeline that counts real-time vehicle density at intersections using custom-trained models. Integrates a predictive emergency vehicle routing override system.
* **Stack:** `Next.js` · `Flask` · `YOLOv4` · `OpenCV`

### 🏍️ RIGS (Road Infraction Governance System) — *AI Traffic Enforcement*
* **The Core Problem:** Manual monitoring of safety violations scales poorly and is highly prone to human tracking delays.
* **The Engineering Solution:** Developed an automated pipeline that screens continuous video feeds for two-wheel rider helmet compliance. On detection of a violation, the system isolates the vehicle bounding box, extracts the registration vector via OCR, and instantly flags it for an automated e-challan system.
* **Stack:** `React` · `YOLOv8` · `Node.js` · `Custom OCR Engines`

### 📸 Wedding Face Finder — *High-Throughput Biometric Search Engine*
* **The Core Problem:** Sifting through thousands of unorganized, ultra-high-resolution wedding photos to locate a specific individual is labor-intensive.
* **The Engineering Solution:** Engineered a deep biometric indexing system. It extracts distinct facial embeddings from raw bulk directories, maps them mathematically into structured clusters using Face Recognition architectures, and allows immediate sub-second image filtering based on a single reference face query.
* **Stack:** `Flask` · `DeepFace` · `ArcFace` · `SQLite`
